Condicoes aninhadas:

if (cond1) {
    bloco1
} else {
    if (cond2) {
        bloco2
    } else {
        bloco3
    }
}  <-__
        \__  condicao aninhada

Condicao multipla:
switch (expressao) {
    case valor 1:
        bloco
        break
    case valor 2:
        bloco
        break
    case valor 3:
        bloco
        break
    default:
        bloco
        break (opcional)
}
#### sempre utilize o break
